Servidor para evento _utn talk_ de GDG Mendoza
==
>### Descripción
>Servidor basado en asp.net mvc5, con cors activado por medio de librería thinktecture.identitymodel.
> 
>**PRECAUCIÓN** *La configuración de este servidor acepta CORS (Cross-origin resource sharing) de cualquier sitio.*

>### Uso

>* <p>Luego de clonar el proyecto, abrir con 
  [visual studio 2013 community][4],
  y presionar **F5** para ejecutar y ver en explorador por defecto.</p><p>
 **Nota**:
 La primera vez que ejecute el proyecto, [visual studio][4] debería instalar los paquetes que necesita el proyecto automáticamente, en caso
   contrario, ir a herramientas -> nuget -> consola nuget y una vez dentro de la consola ejecutar
  `Update-Package -reinstall`, esto demorará unos instantes porque nuget descargará los paquetes desde internet.</p>
